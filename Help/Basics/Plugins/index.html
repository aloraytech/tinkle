Plugins

 

Plugins are those class which can be call in controller as event.

How Set A Plugin Before Use It ?  

$this->setPlugin('users.myProfile',[\Plugins\Users\Users::class,'myProfile']);

Now How use This Plugin?  

$user = $this->getPlugin('users.myProfile','Amal');
echo $user;

Another Example with Array Value :- 

Set A Plugin  

$this->setPlugin('posts.new',[\Plugins\Posts\Posts::class,'newPost']);

Get The Plugin  

$data =['title' => "Post Title", 'content'=>'This is a nice day.','author'=>'lovelyUser'];
$post = $this->getPlugin('posts.new',$data);
echo $post;

Plugins Post Class  

namespace Plugins\Posts;

     
use Plugins\Plugins;

     
/**
 * Class Posts
 * @package Plugins\Posts
 */
class Posts extends Plugins
{
    
    public function newPost(array $data)
    {
        return "Post Created With : ". implode(',', $data);
    }
    
}

Output  

Post Created With : Post Title,This is a nice day.,lovelyUser
