Eloquent

Eloquent is a Laravel Framework Packadge, Which is most popular easy to use orm solution for Laravel/Database

When you Set Configuration in Your Config/DbConfig/  

getConfig(): array

 

For Eloquent :-  

'db_service_by'=> self::ELOQUENT_DB_HANDLER,

All necessary Classes, Packadges, Parent Class etc automaticaly changed to Laravel Database aka Eloquent.

You can check those changes in your Migration Files, Seeder Files etc.

Only New File Will be created depend on db_service_by. All Old Created Files are remain same,

For use latest settings, you may update your database related files with little DB Fix.
